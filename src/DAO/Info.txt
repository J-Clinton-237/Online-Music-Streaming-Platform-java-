This is the folder that will hold all classes linking the system to the database.

Example of Classes:

        connectDatabase
        addNewUser(name, email, dateRegistered)
        addNewArtist(name, email, dateRegistered, Genre)
        addPlaylist()
        deletePlaylist(playlist ID)
        addSong(song file URL)
        removeSong(song ID)
        getSong(song ID)
    

Alright to get the database in your system, open the database.sql file in the Database folder
Make sure to install the PostgreSQL server its about 300Mb. 
Also, I recommend installing PostgreSQL Extension from microsoft in VScode.
Create a server instance with username:postgres and password:qwertyuiop.
Link the extension to the server by following a tutorial or ChatBot.
Run the sql file and PostgreSQL will execute and create the database.
